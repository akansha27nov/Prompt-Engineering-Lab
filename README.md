# Prompt Engineering Lab

## Overview
the empirical science of prompt engineering by building production-ready prompts through testing, iteration, and refinement.

This lab documents the process of taking highly unstable "Zero-Shot" prompts and iteratively upgrading them using **Few-Shot Prompting**, **Chain-of-Thought (CoT) Reasoning** to achieve output consistency across repeated stress tests.

## What it includes
1. **Jupyter Notebook** - `prompt_engineering_lab.ipynb` is a full notebook with all 3 versions for sentiment analysis, product description generation and data extraction v1 (zero-shot). It also contains the result for 5/10/15 iterations of testing for each version of prompt along with variations and comparision results.
2. **Lab Summary** - `lab_summary.md` include a short summary of what was done and lesson learned.
